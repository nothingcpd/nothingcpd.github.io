Leetcode系列（1）：Set Matrix Zeroes

明年就要面临找工作问题了，互联网行业找工作首先注重基础，算法这一关是必须要过的，所以，开一个leetcode专题，leetcode总共160多道算法题
每天1-3道，坚持写下来，把重要的题目或者算法，记下来，写成博客增强理解，希望能对自己明年找工作带来帮助。

另附：统治世界的十大算法

题目描述

Given a m x n matrix, if an element is 0, set its entire row and column to 0. Do it in place.

题目翻译

给定一个m×n的矩阵，如果其中的某个元素为0，则将其所在的列和所在的行全部设置为0.原地变化

题目分析

此题其实不难，很容想到其实就是遍历矩阵呗，但是难点或者技巧在于如何在原有的矩阵进行操作，而不涉及额外空间开销。显然使用另外一个矩阵进行记录不可行。这是很容易想到使用其中的某一行和某一列来进行标记。于是很自然我选择使用第0行，第0列进行标记。但是在使用之前需要判断第0行和第0列是否需要置0。

步骤分析

1.设置两个布尔型标示符flag_row,flag_col用来保存第0行和第0列是否需要置0,初始值false
2.遍历第0行，如果遇到0，则将flag_row=true
3.同上遍历第0列，如果遇到0，则将flag_col=true
4.遍历剩下的元素即i从1开始，j从一开始，如果遇到某个元素【i,j】为0，则将matrix[i][0]=0,matrix[0][j]=0
5.遍历第0行，遇到matrix[0][j]=0则设置此列上的所有元素为0
6.遍历第0列，遇到matrix[i][0]=0则设置此行上的所有元素为0
7.判断flag_row是否为true，是的话设置第0行元素均为0
8.判断flag_col是否为true，是的话设置第0列元素均为0
9.至此，题目完成

注意点

使用有限空间实现操作时，注意尝试以后空间的重复使用以及标识
